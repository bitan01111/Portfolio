# Accessibility (A11y)

Keyboard and semantics quick checklist:

- Quick navigation tiles are keyboard-focusable (`<button>`).
- Contact form:
  - All fields have associated `<label for="...">`.
  - Submit status uses `role="status"` with `aria-live="polite"`.
  - On validation failure, focus moves to the first invalid field.
- Modals:
  - Press `Esc` to close the Source modal (and AI panel).

Run manual test:
1. Use `Tab` to navigate interactive controls.
2. Use `Enter` / `Space` to activate buttons.
3. Use `Esc` to close open dialogs.
